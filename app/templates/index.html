About Me

My journey into the tech world started years ago while I was working as a buyer for a small manufacturing company. When asked what it would take to have our ERP system automatically send custom reports to our sales staff at the end of each month, I found the cost of outsourcing that work far too expensive.

Persuaded by my employer to find a solution, I rose to the challenge of learning what was required to design these reports in-house. In this time, I learned MySQL and basic programming concepts and software called iReport. As time passed, more customizations were requested, and I eventually found myself using Python and fell in love with the language.

These days, the majority of my time is spent doing similar work in JavaScript within NetSuite's runtime environment.

Current Projects

chromie

A simple utility CLI tool for packaging and maintaining chrome extensions.

The inspiration for this project came from my coworker's frustration zipping and publishing chrome extensions to the chrome web store. It's shining feature is a pack command that bundles everything required into a single .zip that can be uploaded into google's web store.

pylaunch

Lost your TV remote? No big deal.

I have always had an interest in IoT, and when I learned that I could make HTTP requests to my Roku TV, I immediately had a project on my hands. This project is far from complete, but it's features are convenient, and I still use it everytime I sit down to watch Netflix.

Originally intended to be a library, this project audience is other developers that want to discover and control their smart TV using Python. However, it does feature a GUI that can be used by running roku init in the command line.
